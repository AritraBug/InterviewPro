# InterviewPro Progress Tracker

## Project Overview

InterviewPro is a full-stack interview management platform built using:

### Backend

* Java 21
* Spring Boot
* Spring Security
* JWT Authentication
* Spring Data JPA
* MySQL
* Maven

### Frontend

* React
* Vite
* Tailwind CSS
* Axios
* React Router

### UI Theme

* Airy Blue Delight
* Primary Color: #71A5DE
* Background Color: #F5F9FF
